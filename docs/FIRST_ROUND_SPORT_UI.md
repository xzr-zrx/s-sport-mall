# 第一轮体育商城 UI 改造 — SportHub Mall

本轮在不改变订单与支付核心链路的前提下，将原有商品展示页升级为综合体育用品商城。

## 已完成

- 新增 SportHub Mall 品牌视觉、响应式顶部导航、购物车角标与页脚。
- 新增独立商城首页：主视觉、运动分类、热门商品、新品推荐、服务保障。
- 商品列表支持关键词、分类、排序、分页、加载态、空状态和失败重试。
- 商品详情支持图片兜底、库存状态、数量限制、加入购物车和同类推荐。
- 新增 404 页面、登录回跳、路由滚动复位和移动端菜单。
- 商品接口新增分类、搜索、排序和分页查询。
- Flyway V2 新增体育用品分类和 12 条演示商品数据，并将旧数字商品下架。
- 本地 SVG 商品图随项目部署，不依赖第三方图片服务。

## 暂未实现

“立即购买”按钮保留入口，但完整的跳过购物车下单、确认订单和支付体验放在第二轮实现。
