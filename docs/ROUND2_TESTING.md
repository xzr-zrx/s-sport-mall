# 第二轮手动测试清单

建议首次测试合并版时清空旧数据库卷，确保 Flyway V1、V2、V3 全部执行：

```bash
docker compose down -v
docker compose up --build
```

## 1. 注册与商品

1. 打开 `http://localhost:5173`。
2. 注册一个普通账号并登录。
3. 检查首页、商品筛选、商品详情和移动端布局。

## 2. 立即购买

1. 在商品卡片或详情页点击“立即购买”。
2. 确认进入 `/checkout?source=BUY_NOW...`。
3. 填写收货人、联系电话和详细地址。
4. 提交订单，确认跳转到 `/payment/{orderNo}`。
5. 检查订单商品、金额和收货信息。

## 3. 购物车结算

1. 加入两种商品。
2. 在购物车测试全选、取消选择、修改数量和删除确认。
3. 选择部分商品结算。
4. 创建订单后确认只有选中的购物车条目被删除。

## 4. 模拟支付

1. 在支付页点击“打开支付页面”。
2. 在模拟收银台点击“确认支付”。
3. 支付窗口应显示成功并自动关闭或提供返回商城按钮。
4. 原支付页应通过轮询进入支付结果页。
5. 我的订单与订单详情应显示“已支付”。

## 5. 取消与库存恢复

1. 创建一笔新订单但不要支付。
2. 在我的订单或详情页取消。
3. 订单应显示“已取消”。
4. 返回商品页确认库存恢复。

## 6. 安全校验

可在浏览器开发者工具中修改创建订单请求，尝试传入价格或 userId。接口 DTO 不接收这些字段，订单金额和当前用户必须由后端计算和识别。
